Tämä on viikon 2 AWS harjoitus

AWS (Amazon Web Services) on Amazonin tarjoama pilvipalvelualusta, jota käytetään esimerkiksi verkkosivujen, sovellusten ja tietokantojen ajamiseen internetissä. Se mahdollistaa ohjelmien ja palveluiden julkaisun ilman omaa palvelinta – voit vain ladata koodin, ja AWS huolehtii lopusta.
